# ZIP File Password Cracker 🔐

A simple, user-friendly graphical application built with Python and Tkinter to recover passwords from protected ZIP archives using a wordlist-based attack.

![Python Version](https://img.shields.io/badge/python-3.x-blue.svg)

## 🚀 Features

-   **Intuitive GUI**: A clean and simple interface makes it easy to select files and start the process.
-   **Wordlist Attack**: Efficiently attempts to find the password by iterating through a user-provided list of potential passwords.
-   **Real-time Feedback**: The interface displays the current password being tested and provides a final status (success or failure).
-   **Cross-Platform**: Built with Python's standard libraries, it runs on Windows, macOS, and Linux without modification.
-   **Sample Wordlists**: Comes with several pre-made wordlists to help you get started immediately.

## 📋 Prerequisites

To run this application, you only need **Python 3.x** installed on your system.
